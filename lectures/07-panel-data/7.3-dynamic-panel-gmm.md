---

## 1. The problem — a lagged dependent variable breaks FE

Many outcomes are **habit-forming**: the strongest predictor of how much you play
today is how much you played yesterday. That's a **lagged dependent variable** in
the model:

```math
Y_{it} = \rho\,Y_{i,t-1} + \beta X_{it} + \alpha_i + \epsilon_{it}
```

Run fixed effects (§7.1) on this and it **breaks.** The within-transformation
subtracts $\bar Y_i$ from $Y_{i,t-1}$ — but $\bar Y_i$ *contains* $\epsilon_{it}$
(it's an average that includes period $t$), so the demeaned lag becomes
mechanically correlated with the demeaned error. This is **Nickell bias**, and it
does **not** vanish with more units — it's a function of the panel *length* $T$,
so it's crippling in short panels (tracking 10M users over a 14-day trial).

*Verified* (code in §5): true $\rho = 0.6$ over a short panel ($T = 6$), FE
estimates $\rho = \mathbf{0.306}$ — biased downward by nearly half. No amount of
$N$ fixes it.

---

## 2. The fix — first-difference, then instrument

Arellano & Bond (1991): don't subtract the *mean* (which contaminates the lag) —
subtract *yesterday from today*. First-differencing kills $\alpha_i$ (a constant
differences to zero) without averaging in the current error:

```math
\Delta Y_{it} = \rho\,\Delta Y_{i,t-1} + \beta\,\Delta X_{it} + \Delta\epsilon_{it}
```

But $\Delta Y_{i,t-1} = Y_{i,t-1} - Y_{i,t-2}$ still shares the $\epsilon_{i,t-1}$
term with $\Delta\epsilon_{it} = \epsilon_{it} - \epsilon_{i,t-1}$ — it's
**endogenous.** So instrument it with **deeper historical lags** (Lecture 5): the
*level* $Y_{i,t-2}$ (and further back) is correlated with $\Delta Y_{i,t-1}$ but
uncorrelated with the differenced error $\Delta\epsilon_{it}$, since it predates
both. That's a valid instrument, and GMM combines all the available lags
efficiently.

*Verified* (§5): the same short panel where FE gave 0.306, Arellano-Bond (first-
difference, level lag as instrument) recovers $\rho = \mathbf{0.606}$.

**Difference GMM vs. System GMM.** Arellano-Bond is *difference* GMM. When the
series is highly persistent ($\rho$ near 1), the lagged levels become *weak*
instruments for the differences; **Blundell-Bond system GMM** (1998) adds a second
moment condition (lagged *differences* instrumenting the *levels*) and is more
robust there.

---

## 3. When to use it

The signature: **short panel** (massive $N$, small $T$ — 10M users over 14 days)
with a **habit-forming, autoregressive outcome** (logins, playtime, spend). If $T$
is large, Nickell bias is small and plain FE is fine; dynamic GMM earns its
complexity only in the short-panel regime.

**Two mandatory diagnostics** (both are the IV validity checks from Lecture 5,
specialized):

1. **AR(2) test.** The instruments are valid only if the differenced errors have
   *no second-order serial correlation.* (First-order AR(1) in the differences is
   expected and fine; AR(2) is the alarm.)
2. **Hansen J-test.** The over-identification test (Lecture 5 §5.1): are the deep
   lags jointly valid instruments? Watch **instrument count** — too many
   instruments (relative to $N$) makes Hansen over-fit and lie; keep the
   instrument set lean.

## 4. Worked probe: the Roblox login bonus

**Setup.** Measure the effect of a **daily login bonus** $X$ on **hours played
today** $Y$. Gaming is heavily habit-forming — today's play is driven mostly by
yesterday's.

**Why FE fails.** With $Y_{t-1}$ in the model, the within-transformation forces the
lag to correlate with the error (Nickell bias), and the panel is short (a trial
window). The bonus coefficient is contaminated by the habit loop.

**The Arellano-Bond walkthrough:**

1. **First-difference:** $\Delta Y_t = Y_{\text{today}} - Y_{\text{yesterday}}$,
   $\Delta X_t = \text{Bonus}_{\text{today}} - \text{Bonus}_{\text{yesterday}}$ —
   killing the stable player-skill effect $\alpha_i$.
2. **The endogeneity:** predicting $\Delta Y_t$ from $\Delta Y_{t-1}$ shares
   overlapping error terms.
3. **The GMM instrument:** use hours played on *Sunday* ($t-3$) and *Monday*
   ($t-2$) — deep levels — to instrument the change between Tuesday and Wednesday
   ($\Delta Y_{t-1}$). Sunday's play correlates with Tuesday's (habit) but is
   isolated from Wednesday's *fresh* noise.

**The result.** You separate the **causal effect of the bonus** from the **pure
psychological habit loop**, where FE would have blended the two. The bonus's true
lift emerges once the autoregressive momentum is instrumented out.

## 5. Python

```python
import numpy as np

# Panel Y shaped (N, T). Habit-forming: Y_t depends on Y_{t-1}.

# ---- Nickell bias: FE (within) of rho is biased down in short panels ----
Ylag, Ycur = Y[:, :-1], Y[:, 1:]
yl = Ylag - Ylag.mean(1, keepdims=True); yc = Ycur - Ycur.mean(1, keepdims=True)
rho_fe = (yl.ravel() @ yc.ravel()) / (yl.ravel() @ yl.ravel())     # ~0.306 vs true 0.6

# ---- Arellano-Bond: first-difference, instrument dY_{t-1} with the LEVEL Y_{t-2} ----
dY = np.diff(Y, axis=1)
dY_t, dY_lag = dY[:, 1:].ravel(), dY[:, :-1].ravel()
Z = Y[:, :-2].ravel()                                              # level instrument, 2 lags back
rho_ab = (Z @ dY_t) / (Z @ dY_lag)                                # just-identified IV -> ~0.606

# ---- Production: don't hand-roll ----
# R:      plm::pgmm (Arellano-Bond), or the `pdynmc` package
# Python: linearmodels does not ship AB; use `pydynpd` (system/difference GMM)
# Always report: AR(2) p-value (want > 0.05) and Hansen J (want > 0.05), and the
#                number of instruments (keep << N).
```

## 6. Common misreadings

**"Just add the lagged Y as a regressor in my FE model."**
That *is* the Nickell-bias trap — the within-transformation correlates the lag
with the error. Verified: 0.306 vs true 0.6. First-difference and instrument.
§1–2.

**"Nickell bias goes away with more users."**
It's a function of $T$, not $N$. 10M users over 6 days is just as biased as 1,000.
§1.

**"More instruments (deeper lags) is always better."**
Instrument proliferation makes the Hansen J-test over-fit and falsely pass, and
weakens the estimator. Keep the instrument count well below $N$. §3.

**"Difference GMM works for any persistence."**
For highly persistent series ($\rho \to 1$), lagged levels are *weak* instruments
for differences — use Blundell-Bond system GMM instead. §2.

**"Skip the AR(2) and Hansen tests."**
They're the instrument-validity checks — without them you have no evidence the
deep lags are exogenous. Report both. §3.
